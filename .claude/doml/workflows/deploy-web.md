# DoML Deploy Web Workflow

## Purpose
Generate app.py (FastAPI), templates/index.html (Jinja2 prediction form), requirements.serve.txt,
Dockerfile.serve, and docker-compose.serve.yml into src/<modelname>/vN/ — then confirm that
`docker compose -f docker-compose.serve.yml up` starts the inference service on port 8080.

## Invoked by: /doml-deploy-web [--deploy-dir src/<modelname>/vN]

## Workflow

### Step 1 — Validate project state

Read `.planning/STATE.md`. If it does not exist, display and stop:
```
No DoML project found. Run /doml-new-project first.
```

### Step 2 — Locate deployment directory

Parse `--deploy-dir` flag from `$ARGUMENTS`:
```bash
DEPLOY_DIR=""
if echo "$ARGUMENTS" | grep -q -- '--deploy-dir'; then
  DEPLOY_DIR=$(echo "$ARGUMENTS" | sed 's/.*--deploy-dir[[:space:]]\+\([^[:space:]]*\).*/\1/')
fi
```

If `DEPLOY_DIR` is empty, find the most recent deployment directory:
```bash
if [ -z "$DEPLOY_DIR" ]; then
  DEPLOY_DIR=$(python3 -c "
import glob, re, sys, os
dirs = glob.glob('src/*/*')
versioned = [(d, int(re.search(r'/v(\d+)$', d).group(1))) for d in dirs if re.search(r'/v(\d+)$', d) and os.path.isfile(d + '/deployment_metadata.json')]
if not versioned:
    print('')
else:
    latest = sorted(versioned, key=lambda x: x[1], reverse=True)[0][0]
    print(latest)
")
  if [ -z "$DEPLOY_DIR" ]; then
    echo "No deployment directory found. Run /doml-deploy-model first."
    exit 1
  fi
  echo "Using deployment directory: $DEPLOY_DIR"
fi
```

Validate the deployment directory exists and contains deployment_metadata.json:
```bash
if [ ! -f "$DEPLOY_DIR/deployment_metadata.json" ]; then
  echo "deployment_metadata.json not found in $DEPLOY_DIR"
  echo "Run /doml-deploy-model first to scaffold the deployment directory."
  exit 1
fi
```

### Step 3 — Read deployment_metadata.json

```bash
python3 -c "
import json, sys, os

deploy_dir = '${DEPLOY_DIR}'
with open(deploy_dir + '/deployment_metadata.json') as f:
    meta = json.load(f)

model_file   = meta.get('model_file', 'models/best_model.pkl')
model_name   = meta.get('model_name', 'model')
version      = meta.get('version', 'v1')

feature_schema = meta.get('feature_schema', [])
problem_type_meta = meta.get('problem_type', '')

if target != 'web_service':
    print(f'ERROR: deployment_metadata.json target is \"{target}\", not \"web_service\".')
    print('This deployment was not created for the web service target.')
    print('Run /doml-deploy-model and select \"Web service (FastAPI + Docker)\" to create a web deployment.')
    sys.exit(1)

with open('/tmp/doml_web_meta.sh', 'w') as f:
    f.write(f'MODEL_FILE=\"{model_file}\"\n')
    f.write(f'MODEL_NAME=\"{model_name}\"\n')
    f.write(f'VERSION=\"{version}\"\n')
    f.write(f'PROBLEM_TYPE_META=\"{problem_type_meta}\"\n')

print(f'Model: {model_name} ({model_file}), version: {version}')
print(f'Feature schema: {len(feature_schema)} features')
"
if [ $? -ne 0 ]; then exit 1; fi
source /tmp/doml_web_meta.sh
```

Read problem_type with fallback to config.json:
```bash
PROBLEM_TYPE="${PROBLEM_TYPE_META}"
if [ -z "$PROBLEM_TYPE" ]; then
  PROBLEM_TYPE=$(python3 -c "import json; c=json.load(open('.planning/config.json')); print(c.get('problem_type','regression'))")
  echo "problem_type not in metadata — read from config.json: $PROBLEM_TYPE"
fi
echo "Problem type: $PROBLEM_TYPE"
```

### Step 6 — Generate app.py

```bash
export DEPLOY_DIR MODEL_NAME VERSION PROBLEM_TYPE
python3 << 'PYEOF'
import json, os

deploy_dir   = os.environ['DEPLOY_DIR']
model_name   = os.environ['MODEL_NAME']
version      = os.environ['VERSION']
problem_type = os.environ['PROBLEM_TYPE']

with open(os.path.join(deploy_dir, 'deployment_metadata.json')) as f:
    meta = json.load(f)

app_py = f'''#!/usr/bin/env python3
"""
DoML web service — generated by /doml-deploy-web
Model: {model_name}
Deployment: {version}
"""
import json
from pathlib import Path
from typing import Any

import joblib
import pandas as pd
from fastapi import FastAPI, Request
from fastapi.responses import HTMLResponse, JSONResponse
from fastapi.templating import Jinja2Templates
from pydantic import create_model

# ── Bootstrap ────────────────────────────────────────────────────────────────
_HERE = Path(__file__).parent
with open(_HERE / "deployment_metadata.json") as _f:
    _metadata = json.load(_f)

MODEL_NAME     = _metadata["model_name"]
PROBLEM_TYPE   = _metadata.get("problem_type", {json.dumps(problem_type)})
FEATURE_SCHEMA = _metadata["feature_schema"]
VERSION        = _metadata.get("version", {json.dumps(version)})

# Dynamic Pydantic request model (D-02)
_TYPE_MAP = {{"int64": float, "float64": float, "object": str}}
_fields = {{
    feat["name"]: (_TYPE_MAP.get(feat["type"], str), ...)
    for feat in FEATURE_SCHEMA
}}
PredictRequest = create_model("PredictRequest", **_fields)

# Load model
_model = joblib.load(_HERE / "model.pkl")

# Jinja2 templates
_templates = Jinja2Templates(directory=str(_HERE / "templates"))

# ── App ───────────────────────────────────────────────────────────────────────
app = FastAPI(title=MODEL_NAME, version=VERSION)


@app.get("/health")
async def health() -> dict:
    return {{"status": "ok", "model_name": MODEL_NAME, "version": VERSION}}


@app.get("/schema")
async def schema() -> dict:
    return {{"feature_schema": FEATURE_SCHEMA}}


@app.get("/", response_class=HTMLResponse)
async def form(request: Request):
    return _templates.TemplateResponse("index.html", {{
        "request":      request,
        "model_name":   MODEL_NAME,
        "features":     FEATURE_SCHEMA,
        "problem_type": PROBLEM_TYPE,
    }})


@app.post("/predict")
async def predict(body: PredictRequest) -> JSONResponse:
    df  = pd.DataFrame([body.model_dump()])
    raw = _model.predict(df)[0]
    pred_val = raw.item() if hasattr(raw, "item") else raw
    result: dict[str, Any] = {{"prediction": pred_val}}

    if PROBLEM_TYPE in ("classification", "binary_classification", "multiclass_classification"):
        if hasattr(_model, "predict_proba"):
            proba   = _model.predict_proba(df)[0]
            classes = _model.classes_
            result["probabilities"] = {{
                str(cls): float(p) for cls, p in zip(classes, proba)
            }}

    return JSONResponse(result)
'''

out_path = os.path.join(deploy_dir, 'app.py')
with open(out_path, 'w') as f:
    f.write(app_py)
print(f'Generated: {out_path}')
PYEOF
if [ $? -ne 0 ]; then
  echo "Failed to generate app.py"
  exit 1
fi
```

## Anti-Patterns (do NOT do these)

- NEVER mount `./data/` in docker-compose.serve.yml — the service is self-contained from deployment_metadata.json
- NEVER hardcode field definitions in app.py — always use pydantic.create_model() from feature_schema
- NEVER use feature_schema as the input DataFrame directly — call body.model_dump() to get a dict first
- NEVER build or run the service — only generate files; user runs docker compose up themselves
- NEVER omit `COPY deployment_metadata.json .` from Dockerfile.serve — app.py reads it at startup
- NEVER use shell-interpolated user input in python3 heredocs — DEPLOY_DIR is validated via file existence check before use
