<objective>
Run the DoML Modelling phase end-to-end for any problem type:

Supervised (regression / classification):
1. Copy and execute preprocessing notebook (notebooks/03_preprocessing.ipynb)
2. Copy and execute modelling notebook (notebooks/04_modelling_{type}_v{N}.ipynb)
3. Write interpretation cell and executive narrative
4. Generate reports/model_report.html

Unsupervised (clustering / dimensionality_reduction):
1. Copy and execute unsupervised modelling notebook
2. Write interpretation cell
(HTML report generated by a later dedicated reporting command)

Optional: pass --guidance "your direction" to provide analyst context that shapes
the interpretation cells written after execution.
</objective>

<execution_context>
@.claude/doml/workflows/modelling.md
</execution_context>

<context>
Guidance (optional): $ARGUMENTS
</context>

<process>
Execute the modelling workflow from @.claude/doml/workflows/modelling.md end-to-end.
</process>
