<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI argument to workflow | User supplies `--model <file>` path via $ARGUMENTS |
| Workflow to filesystem | Workflow creates directories and writes JSON under `src/` |
| Workflow to model pkl | Workflow loads model pkl via joblib to derive class name |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-01 | Tampering | --model argument parsing | mitigate | Validate path with `os.path.exists()` before use; use pathlib for all path construction; never shell-interpolate user-supplied string via `$(...)` or `eval` — only pass to Python `os.path.exists()` |
| T-13-02 | Tampering | Directory creation path | mitigate | Use `pathlib.Path.mkdir(parents=True, exist_ok=True)` only; path built from sanitized model_slug (alphanumeric + underscore only after re.sub); never concatenate raw user input into shell mkdir |
| T-13-03 | Tampering | joblib.load() deserialization | accept | pkl files are analyst-controlled project artifacts; no external input path; equivalent risk to running any Python script in the project |
| T-13-04 | Information Disclosure | deployment_metadata.json | accept | Contains only model metadata (feature names, dtypes, model name); no credentials, API keys, or PII; project-local file |
| T-13-05 | Tampering | in-place model_metadata.json update | mitigate | Read full JSON, add only `model_name` key, write back — never delete or overwrite other keys; pattern is `meta["model_name"] = value` then `json.dump(meta, f, indent=2)` |
</threat_model>

**### Step 4 — Ensure model_name in model_metadata.json (DEPLOY-05)**

```bash
python3 -c "
import json, os, joblib

meta_path = 'models/model_metadata.json'
model_file = '${MODEL_FILE}'

if not os.path.exists(meta_path):
    meta = {'model_file': model_file}
else:
    with open(meta_path) as f:
        meta = json.load(f)

if 'model_name' not in meta:
    model = joblib.load(model_file)
    # Drill into Pipeline to get the final estimator class name (not 'Pipeline')
    if hasattr(model, 'steps'):
        estimator = model.steps[-1][1]
        raw_name = type(estimator).__name__
    else:
        raw_name = type(model).__name__
    meta['model_name'] = raw_name
    with open(meta_path, 'w') as f:
        json.dump(meta, f, indent=2)
    print(f'model_name added to model_metadata.json: {raw_name}')
else:
    print(f'model_name already present: {meta[\"model_name\"]}')
"
```

**### Step 5 — Derive MODEL_SLUG**

```bash
MODEL_SLUG=$(python3 -c "
import json, re
with open('models/model_metadata.json') as f:
    meta = json.load(f)
raw_name = meta.get('model_name', 'model')
# Strip parentheses and their contents: 'XGBoost (tuned)' -> 'XGBoost tuned'
slug = re.sub(r'\([^)]*\)', '', raw_name)
# Replace non-alphanumeric runs with underscore
slug = re.sub(r'[^a-zA-Z0-9]+', '_', slug)
# Lowercase and strip leading/trailing underscores
slug = slug.strip('_').lower()
print(slug)
")
echo "Model slug: $MODEL_SLUG"
```

**### Step 7 — Scan for existing versions and set VERSION (D-02)**

```bash
VERSION=$(python3 -c "
import glob, re
model_slug = '${MODEL_SLUG}'
existing = glob.glob(f'src/{model_slug}/v*')
versions = [
    int(re.search(r'/v(\d+)$', d).group(1))
    for d in existing
    if re.search(r'/v(\d+)$', d)
]
print(max(versions) + 1 if versions else 1)
")
echo "Deployment version: v${VERSION}"
```

If `VERSION` is greater than 1, announce before creating (D-02):

if [ "$VERSION" -gt 1 ]; then

**### Step 8 — Read feature_schema**

```bash
FEATURE_SCHEMA=$(python3 -c "
import json, glob, os

# Primary: read preprocessed dataset column dtypes (header-only for speed)
processed_files = glob.glob('data/processed/preprocessed_*')
if processed_files:
    import pandas as pd
    df = pd.read_csv(processed_files[0], nrows=0)
    schema = [{'name': col, 'type': str(df[col].dtype)} for col in df.columns]
else:
    # Fallback: use feature_names from model_metadata.json with unknown type
    with open('models/model_metadata.json') as f:
        meta = json.load(f)
    feature_names = meta.get('feature_names', [])
    schema = [{'name': name, 'type': 'unknown'} for name in feature_names]

print(json.dumps(schema))
")
```

**### Step 9 — Create deployment directory and write deployment_metadata.json (DEPLOY-04)**

```bash
python3 -c "
import json, os
from pathlib import Path
from datetime import datetime, timezone

model_file = '${MODEL_FILE}'
model_slug = '${MODEL_SLUG}'
version = ${VERSION}
target_key = '${TARGET_KEY}'
feature_schema = ${FEATURE_SCHEMA}

with open('models/model_metadata.json') as f:
    meta = json.load(f)
model_name = meta.get('model_name', model_slug)

deploy_dir = Path(f'src/{model_slug}/v{version}')
deploy_dir.mkdir(parents=True, exist_ok=True)

metadata = {
    'model_file': model_file,
    'model_name': model_name,
    'target': target_key,
    'build_date': datetime.now(timezone.utc).isoformat(),
    'version': f'v{version}',
    'feature_schema': feature_schema
}

with open(deploy_dir / 'deployment_metadata.json', 'w') as f:
    json.dump(metadata, f, indent=2)

print(f'Created: src/{model_slug}/v{version}/deployment_metadata.json')
"
```
