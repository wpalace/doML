<objective>
Extend `deploy-model.md` Steps 8 and 9 to write an enriched `feature_schema` and include
`problem_type` in `deployment_metadata.json`.

The web service target (Phase 15) requires two additional fields per feature:
- `categories`: sorted unique string values (for object/categorical columns) or `null` (numerics) — populates `<select>` dropdowns in the prediction form (WEB-06)
- `example`: first-row value as a string — used as `placeholder` text in numeric inputs and in `GET /schema` (WEB-04)

It also requires `problem_type` in `deployment_metadata.json` so `app.py` can decide
whether to call `predict_proba()` and whether to show probability bars.

This is a backward-compatible extension — existing fields (`model_file`, `model_name`, `target`,
`build_date`, `version`) are untouched. The extended schema shape is:
```json
[{"name": "age", "type": "float64", "example": "34.0", "categories": null},
 {"name": "region", "type": "object", "example": "EU", "categories": ["APAC", "EU", "NA"]}]
```
</objective>

```python
import json, glob, os

processed_files = sorted(glob.glob('data/processed/preprocessed_*'))
if processed_files:
    import pandas as pd
    df = pd.read_csv(processed_files[0])
    first_row = df.iloc[0] if len(df) > 0 else None
    schema = []
    for col in df.columns:
        dtype_str = str(df[col].dtype)
        entry = {'name': col, 'type': dtype_str}
        # Example value from first data row
        if first_row is not None:
            val = first_row[col]
            entry['example'] = str(val.item() if hasattr(val, 'item') else val)
        else:
            entry['example'] = '0' if ('int' in dtype_str or 'float' in dtype_str) else 'value'
        # Categories for object (categorical) columns; null for numerics
        if dtype_str == 'object':
            cats = sorted(df[col].dropna().unique().tolist())
            entry['categories'] = [str(c) for c in cats]
        else:
            entry['categories'] = None
        schema.append(entry)
else:
    with open('models/model_metadata.json') as f:
        meta = json.load(f)
    feature_names = meta.get('feature_names', [])
    schema = [{'name': name, 'type': 'unknown', 'example': 'value', 'categories': None} for name in feature_names]
print(json.dumps(schema))
```
