**12e — Write the Claude narrative and inject it via nbformat:**

Based on the benchmark results extracted in Step 12d, write a 2–3 paragraph performance summary covering:
- Single-prediction latency in plain language (fast/moderate/slow relative to typical ML serving thresholds)
- Batch scalability commentary (does latency-per-row decrease at scale?)
- Throughput projection and what it means for the use case
- Parity test confirmation
- Any target-specific caveats (e.g., "ONNX benchmark is a Python approximation; browser WASM may vary")

Then write this Python script to `/tmp/doml_insert_report_summary.py`:

```python
import nbformat, uuid

NARRATIVE = """## Performance Summary

[CLAUDE WRITES 2-3 PARAGRAPHS HERE based on extracted metrics]
"""

with open('notebooks/deployment_report.ipynb') as f:
    nb = nbformat.read(f, as_version=4)

cell = nbformat.v4.new_markdown_cell(source=NARRATIVE)
cell['id'] = uuid.uuid4().hex[:8]
nb.cells.insert(0, cell)

with open('notebooks/deployment_report.ipynb', 'w') as f:
    nbformat.write(nb, f)

print('Narrative injected at cell 0')
```

Run the script:
```bash
python3 /tmp/doml_insert_report_summary.py
```
