Write `/home/bill/source/DoML/README.md` with the exact section order and content described below.

#### Section 1: Title + one-line description

```markdown
# DoML — Do Machine Learning

A Claude Code framework for reproducible, end-to-end ML analysis — from raw data to deployed model.
```

#### Section 2: What is DoML?

3 paragraphs covering:
- DoML is a collection of Claude Code slash commands and workflow templates that guide data scientists through the full ML lifecycle: business framing, EDA, modelling, optional forecasting, and deployment — producing reproducible Jupyter notebooks and stakeholder HTML reports at each phase.
- It enforces reproducibility rules automatically: random seeds, relative paths via `PROJECT_ROOT`, immutable raw data, pinned `requirements.txt`, and `nbstripout` pre-commit hooks.
- Every command is context-aware: it reads `config.json` (problem type, language preference, time factor) so you never answer the same question twice.

#### Section 3: Requirements

- Docker Desktop (or Docker Engine on Linux)
- Claude Code CLI — `npm install -g @anthropic-ai/claude-code`  
  _or_ GitHub Copilot with agent mode enabled in VS Code
