# Plan 20-01: `install.sh` — Bash Install Script

## Goal

Create `install.sh` at the DoML repo root. Downloads the DoML framework archive from GitHub, extracts it to a temp directory, cherry-picks `.claude/skills/doml-*/`, `.claude/doml/workflows/`, `.claude/doml/templates/`, and `CLAUDE.md` into the user's CWD, creates empty `data/` subdirectories, and cleans up the temp directory on any exit.

## File Created

- `install.sh` (repo root)

## Decisions Applied

- D-01: Single `.tar.gz` archive download (not individual file curls)
- D-03: Install into CWD; no arguments
- D-04: Skills, workflows, templates, CLAUDE.md, data dirs
- D-05: `VERSION` env var → `refs/tags/$VERSION`; default → `refs/heads/main`
- D-06: Skills/workflows/templates always overwrite; CLAUDE.md skipped if exists; data dirs never deleted
- D-07: Fail-fast with URL in error message; temp dir cleaned up before exit
- D-08: CLAUDE.md installed from repo root as-is

## Tasks

1. Write `install.sh` with `#!/usr/bin/env bash` shebang, `set -euo pipefail`, VERSION URL construction, `mktemp -d` temp dir with `trap cleanup EXIT`, curl download with fail-fast, tar extraction with root dir validation, skill glob loop, workflows copy, templates recursive copy, CLAUDE.md conditional copy, data dir `mkdir -p`, success message.
2. `chmod +x install.sh`
