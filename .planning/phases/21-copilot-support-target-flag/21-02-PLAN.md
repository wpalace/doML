| Boundary | Description |
|----------|-------------|
| User CLI → install script | --target / -Target value crosses here; attacker-controlled string |
| install script → file system | Script writes to user's working directory; path traversal possible via archive content |
| HTTPS → local archive | Downloaded archive is extracted; MITM could inject files (out of scope, Phase 20 pattern) |

| T-21-02-04 | Tampering | AGENTS.md missing from archive | mitigate | Explicit `test -f "$SRC/AGENTS.md"` guard before copy, exits 1 with clear error if missing — already in action block |
| T-21-02-05 | Denial of Service | CLAUDE.md unconditional overwrite (D-06) | accept | User intentionally re-runs the installer to upgrade; overwriting project instructions is the expected behavior; user is warned via README Quick Start |
